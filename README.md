# 🎓 BunkBuddy — Your Smart Academic Assistant

BunkBuddy is a web application that helps students 
manage their attendance, predict CGPA, and get 
smart alerts about their academic habits.

## ✨ Features
- 📅 **Bunk Calculator** — Know exactly how many 
classes you can skip safely
- 🎓 **CGPA Predictor** — Predict your GPA based 
on your lifestyle and scores
- ⚠️ **Smart Alerts** — Get real-time warnings 
about distractions and attendance

## 🛠️ Tech Stack
- Python, Flask
- Scikit-learn (Linear Regression)
- HTML, CSS

## 📊 ML Model
- Trained on real student dataset from Kaggle
- R2 Score: 85%
- Features: Attendance, Study Hours, Sleep, 
Stress Level, Internal Marks
